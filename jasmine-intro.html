Important Keywords and Functions

  • Describe - Given to us by Jasmine, and used to organize our tests. "Let me describe ____ to you."
  • It - Used inside the describe functions. "Now that I've described something, let me tell you more about ____
  • Expect - Lives inside the it function, and makes expectations/assertions about the it. "Here's what I expect."

Example Conceptual Exercise

  • Describe("earth")
    • It("Is round")
      • Expect(earth.isRound.toBe(true)
    • It("3rd planet from the sun")
      • Expect(earth.numberFromSun)toBe(3)

Matchers

  1. toBe / not.toBe: Compares with "==="
  2. toBeCloseTo: Compares two values using a third for precision
  3. toBeDefined: Ensures variables are not undefined
  4. toBeTruthy/toBeFalsey: Used when expecting boolean values
  5. toBeGreaterThan / toBeLessThan
  6. toContain: Useful with arrays. Can be used to test if a value is inside an array.
  7. toEqual: Compares two objects and their contents, even if they are different references in memory. Unlike toBe.
  8. jasmine.any(): Not a matcher, but will check the type of something for arrays and functions.
